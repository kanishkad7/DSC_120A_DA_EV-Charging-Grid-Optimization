# DSC_120A_DA_EV-Charging-Grid-Optimization
The rapid rise of electric vehicles (EVs) is reshaping energy consumption patterns and placing new demands on electrical grids. 
Understanding how, when, and where EV charging occurs is essential for ensuring grid stability and planning future infrastructure. This project analyzes EV charging and grid optimization data to uncover behavioral patterns, identify stress points, and support data‑driven decision‑making for sustainable grid management.
